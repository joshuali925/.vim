---
description: Commit changes using conventional commits format
---

Commit ONLY your changes to git, do not add pre-existing unstaged diffs. Use the `--signoff` flag. Use conventional commits format with no emojis, keep the commit title simple and high level, not implementation details. Don't add descriptions unlesss necessary.

$ git log --oneline -n 5
!`git log --oneline -n 5`

$ git status
!`git status`
